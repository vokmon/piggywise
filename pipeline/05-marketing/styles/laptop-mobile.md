# Style: laptop-mobile

Dark premium aesthetic. Cover shows both laptop and phone frames side by side. All other images use laptop frame only. Clean, minimal text — product name and one-line feature callouts only. No price in any image.

---

## Templates folder

The agent browses these Canva folders and shows thumbnails so you can pick a template interactively. Add designs to the relevant folder to make them available.

| Asset | Canva folder name |
|---|---|
| Cover | `templates/laptop phone mockup` |
| Preview images | `templates/laptop phone mockup` |
| Video | `templates/laptop phone mockup` |

---

## Background

- Color: dark gradient — `#0F0F1A` (top) to `#1A1A2E` (bottom)
- No texture, no pattern

## Device frames

- **Cover** — laptop frame (left, larger) + phone frame (right, smaller) side by side
- **All previews** — laptop frame only
- Frame style: minimal clay-style, light gray or white

## Asset requirements

- Cover needs two shots: one `cover-desktop.*` (for laptop frame) and one `cover-mobile.*` (for phone frame)
- All other screenshots → preview images, laptop frame only

## Typography

- Product name: bold sans-serif, large, white (`#FFFFFF`)
- Tagline / caption: light sans-serif, small, muted (`#AAAACC`)
- No more than 2 lines of text per image
- No price, no currency

## Accent

- Use `#E94560` for subtle highlights — thin underline, icon dot, or caption chip

## Cover layout

- Laptop mockup left-center (larger), phone mockup right-center (smaller), slight overlap
- Product name top-left, large
- Tagline below product name, small

## Preview layout

- Laptop frame centered
- Caption text bottom-left, small
- Generous padding around the frame (min 15% of canvas width each side)

## Video layout

- First page: product name centered, large — no device frame
- Subsequent pages: footage clip full-bleed, feature callout text bottom-third overlay
- Transitions: simple fade
