# PiggyWise

Tech tools for non-techies — practical Google Sheets templates and AI-powered tools, built for real work.

## Structure

```
piggywise/
  products/              ← product lines
    smart-portfolio/     ← AI-powered investment portfolio tracker (Google Sheets)
  skills/                ← shared Claude Code skills across products
```

## Products

### Smart Portfolio
Google Sheets-based portfolio management system with AI integration. Three strategies: Aggressive Go (stocks + crypto), Hybrid Wealth (growth + dividends), Retire Smooth (long-term retirement). Available in Thai (Fastwork) and English (Etsy).

See [`products/smart-portfolio/README.md`](products/smart-portfolio/README.md) for full details.
