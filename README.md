# PiggyWise (หมูทอง)

**AI-Powered Portfolio Management System for Thai Investors**

PiggyWise sells Google Sheets templates that act as a personal finance coach: automated monthly rebalancing (DCA), psychological guardrails, and an AI Prompt Library that turns any AI into the customer's personal financial advisor. All data stays in the customer's own Google Drive — PiggyWise has no backend.
