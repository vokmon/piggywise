# activity-2-3-pattern-recognition

Generate Pattern Recognition Grids for Set 2: Focus & Concentration.

## Input
- Theme name (e.g., "1970s")
- 10 pattern types for the theme

## Output
- 10 Pattern Recognition Grid sets
- Each set: 5–7 individual pattern challenges
- Answer key with explanation for each pattern
- Saved to: `workspace/[THEME]/activities/2-3-pattern-recognition.md`

## Steps
1. Open `docs/prompt_library.md` and copy the Activity 2.3 prompt template
2. Replace `[THEME]` and `[PATTERN TYPE 1–10]`
3. Run the prompt in Claude/ChatGPT
4. Save full output to `workspace/[THEME]/activities/2-3-pattern-recognition.md`

## Quality Check
- Patterns are logical and rule-based (not arbitrary)
- Each pattern has exactly one correct answer
- Answer explanation clearly states the rule
- Mix of number sequences, letter patterns, and logical progressions
- Inspired by or themed around the era (e.g., years, chart positions, TV episode numbers)

## Tone & Presentation
See `skills/activity-tone.md` for full guidelines. Activity-specific notes:

- **Game name format:** "Spot the Pattern: [Topic]" — e.g., "Spot the Pattern: 1970s Chart Hits"
- **Intro line:** e.g., "These sequences all have a hidden rule. Can you figure out what comes next?"
- **Instructions:** "Here's how to play: Look at each sequence and figure out the pattern. Write the next number, letter, or word in the blank. Take your time — the answer is always logical!"
- **Difficulty label:** Top right of page
- **Score box:** "How did I do? ___ / [N]" — bottom of page
- **No reflection prompt** — logic activity, skip it
