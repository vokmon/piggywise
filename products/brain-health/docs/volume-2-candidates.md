# Brain Health Activity Pack — Volume 2 Candidates

Volume 1 ships with 9 activities. These are the evaluated candidates for Volume 2.

## Top 3 Picks

| Activity | Why |
|----------|-----|
| **KenKen / Calcudoku** | Sudoku-style grid with arithmetic cages — combines spatial + arithmetic thinking. Widely used in senior puzzle books, strong credibility. |
| **Kakuro** | Number crossword where each across/down run must sum to a target using unique digits 1–9. Print-native, self-checking, very satisfying. |
| **Nonogram / Picross / Griddlers** | Fill grid cells from row/column number clues to reveal a pixel picture. Era-themed images (record player, TV set, etc.) give a visual payoff — totally different feeling from any Volume 1 activity. Strong differentiator. |

## Others Evaluated

| Activity | Verdict | Reason |
|----------|---------|--------|
| Crossmath | Maybe later | Good, but overlaps with Crossnumber in feel. Consider if Crossnumber underperforms. |
| Game 24 | Skip | Given 4 numbers, make 24 — can feel frustrating with no grid to fill and non-obvious solutions. |
| Nerdle | Skip | Math Wordle — core mechanic is colour-coded feedback (green/yellow/grey) which breaks completely in B&W print. |
| Rullo | Filler only | Cross out numbers so rows/columns hit target sums — very simple. Good warm-up but not a full activity. |
| Star Battle | Skip | Too niche, not well known enough for a general senior audience. |
| Str8ts | Skip | Like sudoku but with consecutive runs — more complex, less well known. Learning curve too steep. |
| Logic Grid Puzzles | Already in Vol 1 | Activity 3.1 |
| Sudoku | Already in Vol 1 | Activity 2.3 |
